# IT Support Troubleshooting Lab

El objetivo es demostrar habilidades prácticas en:

• Diagnóstico de problemas de red
• Resolución de fallos en sistemas operativos
• Administración básica de sistemas
• Uso de herramientas de troubleshooting
• Documentación técnica de incidentes

Tecnologías y herramientas utilizadas:

- Windows 10 / Windows 11
- Linux (Ubuntu)
- Comandos de red
- PowerShell
- Bash
- Diagnóstico de servicios del sistema
- Logs del sistema

Escenarios incluidos:

1. Equipo sin conexión a internet
2. Problema de resolución DNS
3. Servicio del sistema detenido
4. Alto consumo de CPU
5. Problema de permisos de usuario
